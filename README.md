# DSH ETL Search AI 2025

Dataset search and discovery system for the DSH RSE coding evaluation.

## Project Overview

Building a system to:
- Extract metadata from CEH Catalogue Service
- Store in a database
- Enable semantic search using AI embeddings
- Provide a web interface for dataset discovery

## Structure

```
dsh-etl-search-ai-2025/
├── backend/     # Python backend (ETL + API + Search)
├── frontend/    # Web interface
└── docs/        # Documentation and AI conversations
```
